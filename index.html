Karpathy 的「LLM 知识管家」工作流 — 把 AI 从问答工具升级为知识架构师

👤 @karpathy | 2026.04.04 🛠️ 可复用方法 💡 工作流优化
核心方法 / 技巧
  • 去中心化 RAG 方案：不依赖复杂的检索系统，直接将所有知识内化为结构化 Markdown 文档（~100篇文章 / ~40万词规模无需 RAG）
  • 三步闭环流程：原始素材收集（Obsidian Web Clipper）→ LLM 自动编译（摘要+概念页+反向链接）→ 持续维护（一致性检查+幻灯片输出+图表渲染）
  • 输出回流机制：新产出自动回到知识库，形成复利效应，越用越值钱
  • 成本极低：约 $0.01/千 token，个人使用几乎可忽略
  • 推荐技术栈：Obsidian + Web Clipper + GPT-4/Claude + Marp（幻灯片）
💡 为什么有用

你已经有 Obsidian 大脑了！这套方法可以把你现有的碎片笔记升级为自维护的知识系统——AI 自动帮你做摘要、建链接、查矛盾、出幻灯片。从"信息消费者"变成"知识架构师"，对一人公司内容矩阵的搭建有直接帮助。

AI Agent 提示词终极指南 — 3C 框架 + 「实习生测试法」

👤 @shumer (Matt Shumer) | 2026.04.21 📝 提示词技巧 🛠️ 可复用方法
核心方法 / 技巧
  • 3C 框架（专为 AI Agent 设计，非普通聊天）：
    Context（上下文+材料）→ Constraints（约束+验证）→ Composition（交付物形状）
  • 「实习生测试法」：把 prompt 交给从未见过你的实习生，他能不回来问你任何问题就开始工作吗？如果能 → Context 够了；如果不能 → Agent 也会猜错
  • 最高杠杆短语：「Don't finish until...」——改变"完成"的定义：从"产生了东西"变为"产生并验证正确"
  • 完整 Prompt 模板：CONTEXT → TASK → CONSTRAINTS → VERIFICATION → OUTPUT FORMAT 五段式结构
  • 4 个实战案例：董事会演示文稿 / 竞品调研 / 内容草稿 / Bug修复
💡 为什么有用

你现在就在用 WorkBuddy 的 Agent 模式！这个框架能让你给 AI 下达的指令从"一句话"变成"完整的工作简报"，大幅减少 AI 猜错方向和交出半成品的情况。特别是"Don't finish until"验证模式，能让 Agent 自主检查产物质量后再交给你。

2026 年 Prompt 框架全景指南 — 从 RTF 到 CREATE，哪个适合你？

📚 综合整理自多篇 X 讨论与学术论文 | 2026.04.19 更新 📝 提示词技巧 💡 工作流优化
6 大实用框架速查
框架复杂度最佳场景
RTF (角色+任务+格式)⭐ 低快速回答、翻译、表格
TASS (任务+受众+风格+结构)⭐⭐ 中文案创作、社媒、邮件 ← 创作者首选
CARE (上下文+行动+结果+示例)⭐⭐ 中商业报告、数据分析
CREATE (人设+请求+示例+细节+格式+负面约束)⭐⭐⭐ 高复杂项目、长文写作
COAST (上下文+目标+行动+场景+任务)⭐⭐⭐ 高战略规划
Chain-of-Thought (思维链)⭐⭐⭐ 高数学、逻辑、解题
立刻能用 3 个技巧
  • 万能迭代法：每次 AI 回复后问"这 3 个弱点是什么？重写它"
  • CREATE + CoT 组合拳：实测邮件回复率从 19% → 64%（提升 3 倍以上！）
  • Few-Shot 省 Token 策略：2-5个高质量示例即可（3 个顶 10 个泛泛之辈）
  • 被严重低估的武器：负面约束（如"不要用反问句开头""不要提竞品"），效果胜过二十行正面指令
💡 为什么有用

你每天都要给 AI 写各种指令（公众号文章、小红书、口播稿、日报）。选对框架 = 输出质量翻倍。TASS 框架特别适合你的内容创作场景——指定受众参数后，同一个任务能自动切换语气和深度。

Claude Design 上线 — 用对话做出专业级 PPT / 原型 / 营销物料

🏢 @AnthropicAI (Mike Krieger 主导) | 2026.04.17 🚀 新工具 💡 工作流优化
核心方法 / 技巧
  • 定位：不是让 AI 画图（那是 Midjourney 的事），而是让 AI 做设计师——产出结构化设计物（PPT/原型/线框图/仪表盘/海报）
  • 5 步闭环工作流：自然语言描述需求 → Opus 4.7 生成初稿 → 4 种迭代方式 → 6 种导出出口（PDF/PPTX/HTML/Canva/ZIP/Claude Code）→ 交付
  • 杀手级能力：Design → Code 闭环——设计稿可直接变成生产级代码
  • Canva 官方合作：Claude Design 做初稿 → 导出到 Canva 精修分发
  • 30 秒拿到想法的视觉呈现，而不是 45 分钟电话会
💡 为什么有用

作为一人公司，你经常需要自己出 PPT、封面图、信息图。Claude Design 能让你用对话快速产出"够用的"专业级设计物。特别是 PPTX 导出功能——以后做课程课件、分享演示可以直接用它出初稿。

⚠️ 注意：Pro 用户约 30 分钟耗尽一周额度（Opus 4.7 token 消耗极高），建议用在重要项目上。

Tool Use Prompting Patterns — 让 AI 调用工具时不再「翻车」

👤 @sureprompts (社区整理) | 2026.04.19 📝 提示词技巧
核心方法 / 技巧
  • 清晰的工具描述：每个工具的功能、输入输出必须精确描述
  • Tool-forcing vs Tool-permitting：区分"必须调用某工具"和"可以选择调用"
  • 错误恢复模式：当工具返回异常时，Prompt 应指导 AI 如何优雅降级或重试
  • 畸形参数处理：明确告诉 AI 遇到不合法输入时的处理策略
💡 为什么有用

你在 WorkBuddy 中大量使用各种工具调用（飞书 CLI、文件读写、搜索等）。这些 Pattern 能让你的 AI Agent 在调用工具时更可靠——减少"调用了错误的工具""传错了参数""遇到错误就卡住"这类问题。
